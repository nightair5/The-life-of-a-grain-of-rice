# AGENTS.md

## 项目身份
这是一个移动端H5互动公益广告项目，名称为《一粒米的一生》。
主题是珍惜粮食、农耕文明、从田间到餐桌。

## 用户身份
用户是网页开发小白，需要代码清晰、步骤明确、方便学习和修改。

## 技术栈
- Vite
- React
- TypeScript
- Framer Motion
- html2canvas
- CSS

## 视觉风格
- 新中式
- 高级公益广告质感
- 米白、稻金、深绿、黛青
- 留白充分
- 不要廉价卡通风
- 不要页面像普通PPT

## 开发规则
1. 每次只执行当前 Spec，不要扩展无关功能。
2. 不要删除已有功能。
3. 新增组件放在 src/components。
4. 页面数据放在 src/data。
5. 样式可以放在 src/index.css 或组件内 className。
6. 图片统一放在 public/images。
7. 所有页面必须适配手机竖屏。
8. 文字必须清晰可读。
9. 每次修改后必须运行 npm run dev 或 npm run build 检查。
10. 完成后说明修改了哪些文件、如何测试、还有什么风险。

## 禁止事项
1. 不要引入复杂后端。
2. 不要登录系统。
3. 不要数据库。
4. 不要大规模重构无关文件。
5. 不要使用真实敏感信息。
